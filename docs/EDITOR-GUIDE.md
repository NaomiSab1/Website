# How to edit your website — 1-page guide

**Where:** `yoursite.com.au/admin` · **Login:** your email + password.

## Edit a page
1. Click **Pages**, then the page you want (e.g. *Home*).
2. The left column lists the page's **sections** in order. Click one to edit it
   on the right — text, images, buttons.
3. Headline trick: wrap a word in asterisks to make it gold italic,
   e.g. `Design. *Develop.* Construct.`

## Change images
- In any image field click **Replace** (or **Choose image…**) → pick from the
  library or **Upload new**. Images are compressed automatically.
- Always fill in **alt text** (a short description) — it helps Google and
  accessibility.

## Rearrange the page
- **Drag** a section by the `⋮⋮` handle (or use the ↑ ↓ arrows).
- **⧉** duplicates a section, **✕** removes it.
- **+ Add section** shows every available block: hero, gallery, testimonials,
  stats, FAQ, contact form, map, video…
- Open **Section layout** at the top of the editor to change a section's
  background (light / warm / dark), spacing, alignment, and width.

## Save → Preview → Publish (important!)
- **Save draft** keeps your work. *Nothing changes on the live site yet.*
- **Preview** shows exactly how the page will look — toggle **Desktop / Mobile**.
- **Publish** puts it live. The website updates within ~30 seconds.

## Made a mistake?
Open the **History** tab inside the page editor and click **Restore** next to an
earlier version. The live site reverts immediately.

## Other things you can edit
- **Projects** — the property portfolio (name, location, beds/baths, status
  *For sale / Sold / Completed*, photos). Pages that show property grids update
  automatically when you publish a project change.
- **Media** — the image library (upload, alt text, delete).
- **Leads** — every contact form submission (also emailed/forwarded if configured).
- **Site settings** (admins only) — menu links, footer, social links, brand
  colours and fonts. Saving here updates the live site immediately.
- **Page & SEO** tab — the page title, URL, and the text Google shows.

## Quick rules of thumb
- Draft first, preview always, then publish.
- Keep headlines short; put detail in body text.
- Don't delete the *Home* page (the system won't let you anyway).
- Forgot your password? Ask an admin to reset it in Supabase, or use
  "Forgot password" if enabled.
