How to manage the site

All changes you make here are saved to the database immediately. To publish changes to the live site, click Publish Changes in the top right — this triggers a rebuild on Vercel (takes ~30 seconds).

  • Properties — Add/edit/remove property listings. Set status to "For Sale" or "Sold".
  • Services — Edit the 4 service descriptions and deliverable lists.
  • Testimonials — Manage client quotes. Only the active testimonial is shown.
  • Stats & Counters — Update the animated numbers on the homepage.
  • SEO — Set page titles and meta descriptions for search engines.
  • Settings — Update contact details, social links, and the Vercel deploy hook.
